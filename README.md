# Secure Blog Management System

A secure Blog Management System developed using PHP and MySQL. This project allows users to register, log in, and manage blog posts with CRUD operations while following basic web security practices.

## Features

- User Registration
- User Login & Logout
- Create Blog Posts
- View Blog Posts
- Update Blog Posts
- Delete Blog Posts
- Search Posts
- Pagination
- Form Validation
- Prepared Statements
- Password Hashing
- Role-Based Access (Admin & User)

## Technologies Used

- PHP
- MySQL
- HTML
- CSS
- XAMPP
- Git & GitHub
